# Proyecto Intermodular: Domótica y Estilo de Vida Saludable 🏠

## Objetivo
El objetivo principal de este repositorio es documentar y albergar el código fuente de nuestro portal web, diseñado para integrar la **domótica del hogar** con una plataforma de **ventas online** enfocada en mantener y mejorar la salud de nuestros usuarios.

## Contexto del Proyecto

### Tipo de Empresa
Se trata de un portal desarrollado para una startup tecnológica centrada en el **bienestar integral** y el Internet de las Cosas (IoT).

### Funcionalidades
* Control centralizado de los diferentes sensores y módulos inteligentes de la casa.
* Catálogo de tienda online con productos enfocados en la nutrición y el deporte.
* Panel de usuario personalizado para el seguimiento de rutinas.

### Alcance y Usuarios Potenciales
El proyecto abarca el diseño web y la integración del sistema. Los usuarios potenciales son aquellas personas que buscan automatizar su entorno doméstico para que les ayude activamente a **mantener hábitos saludables** en su día a día.

## Menú de Navegación
Nuestra web cuenta con la siguiente estructura principal:
- Inicio
- Panel Domótico
- Tienda Online
- Carrito de Compra
- Sobre Nosotros
- Contacto
